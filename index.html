This is a set of test pages for testing page weight tools

  • plain.html - plain HTML file with no linked resources
  • plainhtml_big.html - large plain HTML file with no linked resources
  • plain_with_jpeg.html - plain HTML with single linked JPEG image
  • plain_with_png.html - plain HTML with single linked PNG image
  • plain_with_10_icons.html - plain HTML with 10 small icons
  • plain_with_iframe_plane.html - plain HTML file with plain HTML iframe
  • plain_with_sharing_sharethis.html - page with sharethis.com sharing library added
  • plain_with_sharing_addthis.html - page with addthis.com sharing library added
  • plainhtml_with_analytics.html - page with Google analytics
  • plainhtml_with_youtube.html - page with embedded YouTube video
  • redirect_step1.html - page 1 of a set of 4 designed to redirect to each other (with correctly configured server)
  • plain_with_10_icons_iframe.html - page with icons including iframed page with icons
  • plain_with_ajax_load.html - page that loads image via JavaScript
